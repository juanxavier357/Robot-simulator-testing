## Overview

**The challenge**
Tu misión, si decides aceptarla, es crear el algoritmo que simula el comportamiento de un robot y su testing correspondiente para cada una de las funcionalidades. Recuerda hacer testing a cada movimiento o su correspondiente error.

Para pasar esta misión debes entregar tu código testeado en un 100% y mostrar en una captura el reporte del coverage dado por jest.

También sube la solución a un repositorio y pega el link en la entrega.

Puedes encontrar un repositorio guía aquí: https://github.com/makeitrealcamp/robot-simulator
